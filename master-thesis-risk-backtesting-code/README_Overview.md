# Master thesis risk backtesting code

This repository contains the R code used for the simulation studies and numerical examples in my master's thesis on e-backtesting and betting strategies.

## Repository structure

- `Backtest betting strategies/`  
  Simulation code for GREE, GREL, GREM, WEALTH, COV, and interval-wise procedures.
  
- `Backtests/`  
  E-backtesting for Expectiles, Entropic, OCE, ES-grid and EVaR.

- `Systemic/`  
  Systemic backtesting code for CoVaR, CoES and MES.

- `Tail/`  
  Tail-risk backtesting code for ES, TailSD, RVaR, and median shortfall.

Each subfolder contains:
- one main `.r` script
- one local `README.md` with package requirements and example runs
